# Применение генетического алгоритма для обучения игре в шашки

Репозиторий содержит реализацию системы искусственного интеллекта для игры в шашки, сочетающую нейросетевой подход с эволюционными методами обучения. Проект включает полнофункциональное игровое приложение с графическим интерфейсом.

В основе лежит нейронная сеть прямой связи, веса которой оптимизируются с помощью генетического алгоритма. Процесс обучения моделирует эволюцию популяции из 30 нейросетей: особи играют друг с другом, лучшие отбираются по результатам, скрещиваются и мутируют, передавая удачные комбинации весов следующим поколениям. При принятии решений во время игры используется минимаксный поиск с ограниченной глубиной, где оценка позиций выполняется обученной нейросетью.
